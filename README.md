# Cisco Ethical Hacker Learning Journey

This repository documents my personal learning journey from the Cisco Networking Academy Ethical Hacker course.

The purpose of this repository is to build practical cybersecurity skills, write notes in my own words, document lab practice, and create beginner-friendly security scripts.

## Course Focus

- Ethical hacking fundamentals
- Penetration testing methodology
- Planning and scoping assessments
- Information gathering
- Vulnerability scanning
- Social engineering awareness
- Network security
- Web application vulnerabilities
- Cloud, mobile, and IoT security
- Post-exploitation concepts
- Reporting and communication
- Tools and code analysis

## Repository Structure

- `module-notes/` - Notes from each module written in my own words
- `lab-writeups/` - Practical lab summaries and reflections
- `practical-commands/` - Commands practiced during learning
- `python-tools/` - Beginner Python security tools
- `blogs/` - Weekly learning blogs and reflections
- `reports/` - Report templates and documentation
- `resources/` - Checklists, summaries, and useful references
- `screenshots/` - Allowed screenshots from my learning practice

## Goal

My goal is to develop strong foundations in ethical hacking and cybersecurity through consistent learning, documentation, and practical work.

> Build. Break. Learn. Repeat.
